AngularJS Directives

AngularJS lets you extend HTML with new attributes called Directives. AngularJS has a set of built-in directives which offers functionality to you applications. AngularJS also lets you define your own directives.

  • The ng-app directive initializes an AngularJS application.
  • The ng-init directive initializes application data.
  • The ng-model directive binds the value of HTML controls (input, select, textarea) to application data.
Note!!!

Using ng-init is not very common. You will learn how to initialize data in the chapter about controllers.
